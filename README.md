youChannel
==========

Consente lo scan e visualizzazione dei canali youtube in embed


=================================================================================================================

Per istanziarlo basta semplicemente passare un oggetto con il nome del canale ed uno per le opzioni
```javascript
$('#player').youChannel({
        user:'Basisumisura',
        playopts: {
            autoplay: 1,
            ...
        }
});
```
